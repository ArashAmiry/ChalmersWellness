# Chalmers Wellness

## Prerequisites
Install Maven

## Run the program
1. Download project zip.
2. Extract project.
3. Open command line and navigate to project root folder.
4. Run mvn exec:java

## Troubleshooting
1. run mvn clean and then run mvn exec:java
2. If 1 doesn't work run mvn clean then mvn install and then mvn exec:java
